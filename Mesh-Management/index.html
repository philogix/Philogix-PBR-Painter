Mesh Management

Material set

The Matreial Set window shows all the material from the 3D model in the active object. It allows to add, remove and arrange materials, add Plx Shader to the material. Add-on will work on Plx Shader

Material Slots

1. Material Preview

Show material preview.

2. Material name

Displays the material name, the add-on will be based on the material name to work, so the material name should be set once and not changed during work.

3. Material Status

Displays the status of the material, if the Material Status shows a check icon, the material is ready to work with the add-on.

4. Material Slots Control

Allows adding, removing and moving Material slots.

Plx Shader Control

Plx shader is a Shader created by Philogix PBR Painter add-on. To work with Philogix PBR Painter, the material requires Plx Shader. Plx Shader Control allows adding, removing and Fixing Plx Shader errors on the material.

1. Adding or Fixing Plx Shader

This button allows adding Fix Plx Shader.

2. Remove Plx Shader

Button to remove Plx Shader from the material.

UV set

The UV set window manages the UVs of the active object. You can create multiple UVs to work with but only Export Map on selected UVs.

1. UV slots

Display the UVs of the object

2. UV Control

Button to add or remove UV
