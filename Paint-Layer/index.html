Paint Layer

The Paint Layer lets you manipulate the layers of a Material. A layer contains the painting and effects that will create the texture on the 3D object in the scene. You can hide and unhide layers, put them into folders and change their opacity and blending mode..

The Paint Layer display layers with a specific hierarchy : the layer at the bottom will be drawn first on the mesh, the layer on top will follow. Therefore the layer at the top of the stack is the last item, while the layer at the far bottom is the first one.

Material Layers

Control panel

1. Create new layer

Create a new Material layer above the currently selected one.

Types of layers :

  • Custom Layer: Custom Layer allows building material layer with full channel, can create layers within channel. And control the affected area with Layer Mask.
  • Library Layer: The Library Layer is a pre-made material layer in Philogix PBR Painter that reads information from Bake Maps, processes it, and outputs channels. So you can't add layers in channels but you can control the area with Layer Mask.
2. Import PBR set

Import the PBR set to the Custom Layer.

3. Move layer

Move up or down the selected layer.

4. Remove layer

Delete the currently selected item

5. Viewmode

Viewmode controls the view mode of the Layer paint. The viewmode can be used to defined the current display context. Using this drop-down menu, you can specify which channels are displayed on the object

Material Slots

1. Material Type

Material Type shows material type icon.

2. Mute layer

Muting a layer removes the contribution of the layer to the Material.

3. Name layer

Name layer manages the name of the Material Layer.

4. Clip layer

Clip layer is the layer option that is a sublayer of the layer below. The child layer will only affect the parent layer. The parent layer can have many child layers.

5. Color mark

Color mark has the effect of marking Layers on Layer Paint and in Shader Editor.

Channel Layers

Channel List

Channel List displays a list of the channels of the selected Material Layer, including the Layer Mask.

When a channel in the Channel List is selected, it is displayed in blue. Then the window below will be used to edit that channel.

Channel Settings

Channel Settings controls the parameters of a Channel.

1. Anchor Point

An Anchor Point is a way to expose any resource or element in the Channel List and reference it in different areas of the Channel List for different purposes and with a different set of adjustments. They open up a whole new set of possibilities, allowing you to effectively link layers or masks together and have a single Anchor point affect multiple aspects of your project, transforming Philogix PBR Painter into a truly non-linear experience.

When Anchor Point is activated, you can use this Channel in Anchor Layer

2. Channel Name

Show channel name is working

3. Blending modes

Layers and effects have access to many Blending Modes. They allow to mix the result of a layer with the other layers below in different manners.

4. Channel Opacity

Controls the amount of influence the active channel of the selected Material Layer
