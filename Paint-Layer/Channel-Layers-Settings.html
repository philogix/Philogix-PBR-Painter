Channel Layers Settings

Channel Layers Settings controls the settings of the selected Channel Layer, the Channel Layers Settings panel changes depending on the layer type selected.

UV setting

1. Using UVs

Using UVs is the toggle between Private UV and General UV. If Private UV is selected, the selected layer will use its own UV. If General UV is selected, the selected layer will use the general UV of the Material layer.

2. Projection

Projection to use for mapping the textures.

  • UV:

    The UV option is used to retrieve specific UV maps. This option can retrieve any UV map belonging to the object using the material.
    • UV map: UV map to use.
  • Box:

    Maps the image to the six sides of a virtual box, based on the normal, using XY, YZ and XYZ coordinates depending on the side.
    • Blend: For Box mapping, the amount to blend between sides of the box, to get rid of sharp transitions between the different sides. Blending is useful to map a procedural-like image texture pattern seamlessly on a model. 0.0 gives no blending; higher values give a smoother transition.
  • Flat:

    Uses the XY coordinates for mapping.
  • Sphere:

    Sphere mapping is the best type for mapping a sphere, and it is perfect for making planets and similar objects. It is often very useful for creating organic objects.
3. UV transformation

The UV transformation controls position, rotation and size of the projection.

  • Location: The amount of translation along each axis.
  • Rotation: The amount of rotation along each axis. XYZ order.
  • Scale: The amount of scaling along each axis.

Types of layers :

  • Base Layer:

    Base Layer is the base layer in Philogix PBR Painter, It is valid on the whole object, so the UV setting panel will not be available.

  • Image Layer:

    Image Layer is the layer that contains image information. Here you can use images from the library, bake maps or import images from the outside.

    • 1. Image Preview: Show the preview the image is working.
    • 2. Image Library: Image Library displays a list of libraries:

    - Albedo:

    - Alpha:

    - Bump:

    - Normal:

    - Bake maps:

    • 3. Import Image: Import images from external gallery.
  • Paint Layer:

    Paint the mesh in the 3D Viewport

    • 1. Brush Library: List of paint brushes available in Philogix PBR Painter.
    • 2. Paint Button: Painting start or end button.
    • 3. Tool list: List of drawing tools.
  • Surface Layer:

    Surface Layer is a layer that combines many nodes, it can use bake maps to calculate and process to give the best results.

    • 1. Surface Preview: Show the preview the surface is working.
    • 2. Surface Library: Surface Library displays a list of libraries.
    • 3. Outputs: Each Surface Layer has 2 output options:

      Factor: Value of Surface Layer.

      Color: Color with different Surface Layer in each component.

  • Anchor Layer:

    Anchor Layer uses an Anchor Point enabled Channel.

  • Filter Layer:

    The Filter Layer are a set of various actions that can be applied on the Channel of Philogix PBR Painter.

    They allow an infinite series of changes from simple color variations to complex mask creations.

    Filter Type:

    • Bright/Contrast
    • Hue/Saturation
    • Gamma
    • Invert
    • Black & White
    • Curve
    • Color Ramp
